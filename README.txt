# Web Calculator

The Web Calculator is a simple calculator application built using HTML, CSS, and JavaScript. It provides basic mathematical operations, percentage calculations, power calculations, root calculations, and number system conversions (binary, decimal, hexadecimal). The calculator supports multiple themes, allowing users to customize the visual appearance.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Percentage calculation: Calculate the percentage of a number.
- Power calculation: Raise a number to a given exponent.
- Root calculation: Calculate the square root of a number.
- Number system conversions: Convert numbers between binary, decimal, and hexadecimal.
- Memory function: Store and retrieve values from memory.
- Responsive design: The calculator adapts to different screen sizes and devices.
- Multiple themes: Choose from different visual themes for the calculator.

## Usage

1. Clone or download the project repository.
2. Open the `index.html` file in your web browser.
3. Use the calculator by clicking on the buttons or typing numbers and operators directly.
4. Choose a theme by clicking on the theme buttons at the top of the calculator.
5. Perform mathematical operations, percentage calculations, power calculations, root calculations, and number system conversions using the provided buttons.
6. Clear the display by clicking the "C" button or delete the last character using the backspace button.
7. Use the memory functions "M+" and "MR" to store and retrieve values from memory.

## Development

If you want to modify or enhance the calculator, follow these steps:

1. Fork the repository and clone it to your local machine.
2. Open the project directory in your preferred code editor.
3. Make changes to the HTML, CSS, or JavaScript files as required.
4. Test your changes by opening the `index.html` file in a web browser.
5. Commit and push your changes to your forked repository.
6. Submit a pull request describing the changes you've made.


# Web Számológép

A Web Számológép egy egyszerű számológép alkalmazás, amelyet HTML, CSS és JavaScript segítségével készítettek. Alapvető matematikai műveleteket, százalék számítást, hatványozást, gyökvonást és számrendszer átalakítást (bináris, decimális, hexadecimális) tesz lehetővé. A számológép több témát támogat, amelyek lehetővé teszik a felhasználók számára a vizuális megjelenés testreszabását.

## Funkciók

- Alapvető matematikai műveletek: összeadás, kivonás, szorzás és osztás.
- Százalékszámítás: Számítsa ki egy szám százalékát.
- Hatványszámítás: Egy szám hatványozása adott kitevővel.
- Gyökvonás: Számítsa ki egy szám négyzetgyökét.
- Számrendszer átalakítás: Számok átalakítása bináris, decimális és hexadecimális számrendszer között.
- Memória funkció: Értékek tárolása és visszakeresése a memóriából.
- Reszponzív design: A számológép alkalmazkodik különböző képernyőméretekhez és eszközökhöz.
- Több téma: Válasszon a számológép különböző vizuális témái közül.

## Használat

1. Klónozza vagy töltse le a projekt tárolóját.
2. Nyissa meg az `index.html` fájlt a böngészőjében.
3. Használja a számológépet a gombokra kattintva vagy közvetlenül számokat és operátorokat gépelésével.
4. Válasszon témát a számológép tetején lévő témagombok segítségével.
5. Végezzen matematikai műveleteket, százalékszámítást, hatványszámítást, gyökvonást és számrendszer átalakítást a rendelkezésre álló gombok segítségével.
6. Törölje a kijelző tartalmát a "C" gombbal vagy törölje az utolsó karaktert a backspace gombbal.
7. Használja a memória funkciókat "M+" és "MR" értékek tárolására és visszakeresésére a memóriából.
